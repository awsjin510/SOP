# 前端頁面規格

詳細的頁面結構、元件組成、互動行為。

## 路由設定

```typescript
// src/router/index.ts
import { createRouter, createWebHistory } from 'vue-router';

const routes = [
  // === 公開頁面 ===
  { path: '/', component: () => import('@/pages/LandingPage.vue'), meta: { public: true } },
  { path: '/login', component: () => import('@/pages/LoginPage.vue'), meta: { public: true } },
  
  // === 需登入 ===
  { path: '/dashboard', component: () => import('@/pages/DashboardPage.vue') },
  { path: '/sop/new', component: () => import('@/pages/NewSopPage.vue') },
  { path: '/sop/:id', component: () => import('@/pages/SopDetailPage.vue'), props: true },
  { path: '/sop/:id/update', component: () => import('@/pages/UpdateSopPage.vue'), props: true },
  { path: '/sop/:id/versions/:versionId', component: () => import('@/pages/VersionDetailPage.vue'), props: true },
  { path: '/sop/:id/versions/:versionId/diff/:otherVersionId', component: () => import('@/pages/VersionDiffPage.vue'), props: true },
  { path: '/job/:jobId', component: () => import('@/pages/JobProgressPage.vue'), props: true },
  { path: '/job/:jobId/review', component: () => import('@/pages/ReviewInterfacePage.vue'), props: true },
  { path: '/settings', component: () => import('@/pages/SettingsPage.vue') },
];

const router = createRouter({
  history: createWebHistory(),
  routes,
});

router.beforeEach((to) => {
  if (!to.meta.public && !authStore.isAuthenticated) {
    return '/login';
  }
});
```

**用途**：登入頁（如果使用者點 LandingPage 的登入按鈕後重導）。

**簡單版**：直接觸發 Google 登入彈窗，不單獨的頁面內容。

### SopDetailPage（`/sop/:id`）

**用途**：SOP 詳細頁，多分頁。

**Layout**：
```
┌─────────────────────────────────────────────────────┐
│  EC2 啟動 SOP                       [更新] [📥 下載] │
│                                                      │
│  v1.3.0  最新  ●  AWS 操作  初級  30 分鐘            │
│                                                      │
│  ───────────────────────────────────────────         │
│                                                      │
│  [概覽] [預覽] [版本歷史] [變更紀錄] [素材]            │
│                                                      │
│  ─── 概覽（預設）───                                  │
│  ...                                                 │
└─────────────────────────────────────────────────────┘
```

**5 個分頁**：

#### 概覽（Overview）
- 統計（步驟、troubleshooting、術語、截圖數量）
- 版本歷程（最近 5 次）
- 素材清單

#### 預覽（Preview）
- 線上顯示 SOP 內容（用 Markdown 渲染或 mammoth 轉 HTML）
- 含目錄、可點擊的章節跳轉

#### 版本歷史（Versions）
- 時間軸式呈現所有版本
- 每版本顯示：版本號、日期、作者、變更摘要
- 可點擊跳到特定版本詳細頁
- 可選兩個版本比較

#### 變更紀錄（Changes）
- 列出所有變更（changes collection）
- 每變更顯示：from→to 版本、變更項目數、是否需重訓
- 可下載對應的 changelog.docx

#### 素材（Materials）
- 列出所有版本使用的原始素材
- 可下載
- 顯示素材對哪些 SOP 內容有貢獻

### VersionDetailPage（`/sop/:id/versions/:versionId`）

**用途**：檢視特定版本的內容。
